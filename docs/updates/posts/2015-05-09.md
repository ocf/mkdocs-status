# Another WordPress XSS vulnerability; please update!
<div class="post-body entry-content" id="post-body-8713249893235159805" itemprop="description articleBody" style="background-color: white; color: #222222; font-family: Arial, Tahoma, Helvetica, FreeSans, sans-serif; font-size: 13.1999998092651px; line-height: 1.4; position: relative; width: 570px;">
Another <a href="https://wordpress.org/news/2015/05/wordpress-4-2-2/" style="color: #888888; text-decoration: none;">vulnerability</a> was recently discovered in WordPress which affects a large number of OCF web hosting users. The vulnerability can potentially allow a malicious person to hijack your session and compromise your website.<br /><br />All users should update immediately to the latest version of WordPress. Version 4.2.2 (i.e. the latest version) is the only version we consider safe.<br /><br />Updating WordPress is extremely easy; it's <a href="http://codex.wordpress.org/Updating_WordPress#One-click_Update" style="color: #888888; text-decoration: none;">just a single click</a> after logging in to the admin panel.<br /><br />Recent versions of WordPress come with automatic updates enabled for minor releases, which can help to protect you from future vulnerabilities. We strongly recommend not disabling this feature!<br /><br />If we've contacted you and you need help updating your site, please don't hesitate to <a href="https://ocf.io/contact" style="color: #888888; text-decoration: none;">get in touch</a> so that we can help!</div>

We will be emailing affected users in the near future and offering to upgrade WordPress on their behalf. If you'd like us to not do this, please confirm that either (a) you have updated it yourself, (b) you've removed WordPress entirely, or (c) you'd like to close your OCF account.</div>
