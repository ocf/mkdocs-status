# WordPress XSS vulnerability; please update!
A <a href="https://wordpress.org/news/2014/11/wordpress-4-0-1/">vulnerability</a> was recently discovered in WordPress which affects a large number of OCF web hosting users. The vulnerability can potentially allow a malicious person to hijack your session and compromise your website.<br />
<br />
All users should update immediately to the latest version of WordPress. Versions 3.9.3, 3.8.5, 3.7.5, 4.0.0, 4.0.1 are unaffected by this vulnerability, but we highly advise to always use the latest version.<br />
<br />Updating WordPress is extremely easy; it's <a href="http://codex.wordpress.org/Updating_WordPress#One-click_Update">just a single click</a> after logging in to the admin panel.<br />
<br />
Recent versions of WordPress come with automatic updates enabled for minor releases, which can help to protect you from future vulnerabilities. We strongly recommend not disabling this feature!<br />
<br />
If we've contacted you and you need help updating your site, please don't hesitate to <a href="https://ocf.io/contact">get in touch</a> so that we can help!
