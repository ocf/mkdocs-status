# Downtime this morning - Thursday 4/20 6:53 to 11:15 AM
From around 6:53 to 11:15 this morning a majority of the OCF's services were down, including websites, email forwarding, software mirrors, etc. We take downtime issues like this very seriously, so we are still in the process of doing a full postmortem on exactly what happened and the timeline it followed. Expect that to be posted soon as we finalize the details of what happened, but everything should now be back up and working again. If you notice any problems, please <a href="https://www.ocf.berkeley.edu/docs/contact/">contact us</a> and we'd be happy to help.<br />
