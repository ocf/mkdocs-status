# New Linux Login Server
The OCF has a new Linux login server thanks to a recent hardware donation. This server is configured with the same software as our AMD64 lab machines but is secured in our server room, so it should be more reliable and less-prone to random reboots.<br /><br />Just log into <tt>tsunami.ocf.berkeley.edu</tt> with your OCF account to use this new server.
