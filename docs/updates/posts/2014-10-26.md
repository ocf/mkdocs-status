# Announcing dedicated hosting for web applications!
We're excited to announce a new OCF service for student groups: dedicated hosting for web applications like Rails, Django, Flask, and Node.js!<br />
<br />
Previously, hosting for modern web apps was only available via FastCGI, which was difficult to set up and manage. The new service makes it possible to host any app that can bind to a socket, enabling you to run basically any type of application.<br />
<br />
OCF hosting for web apps is a pretty cool choice for student groups compared to hosting off site; it's easy to get a berkeley.edu domain name and to get support from friendly volunteer staffers during <a href="https://www.ocf.berkeley.edu/staff_hours">staff hours</a>, and now it's easy to set up your app on our powerful servers, all hosted on-campus.<br />
<br />
We're opening up the new application hosting on a trial basis, and hoping to work closely with a small number of student group early adopters to work out kinks and improve the service. If you're interested, take a look at <a href="https://wiki.ocf.berkeley.edu/services/webapps/">the documentation</a> and get in touch!
