# Campus-wide network issues
As of about 9:00am Tuesday morning, Berkeley campus has been experiencing severe network issues, including high packet loss and latency. Open Computing Facility servers are affected by this outage.<br />
<br />
This is a problem on IST's side; updates are available <a href="http://systemstatus.berkeley.edu/">from them</a>.
