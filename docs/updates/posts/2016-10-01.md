# Announcing revamped mail hosting for student groups
We're excited to announce the launch of our new mail service for student groups!<br />
<br />
The OCF has provided mail virtual hosting for many years, allowing student groups to create email addresses such as "{name}@mygroup.berkeley.edu" which forward to their primary email address.<br />
<br />
With the new system, we're introducing a few new features:<br />
<br />
<ul>
<li>Easily edit addresses only. Previously, editing addresses meant modifying hidden <span style="font-family: Courier New, Courier, monospace;">.forward</span> files over SSH. These files are no more!<br /><br />Instead, <a href="https://www.ocf.berkeley.edu/account/vhost/mail">log in to our website</a> and add, edit, and remove addresses from your web browser.<br /><br /></li>
<li>Send or reply to mail. The old system only allowed you to receive mail. We now allow replying to that mail from your student group address (or composing new mail from it).</li>
</ul>
<br />
You can find out more about using the new system <a href="https://www.ocf.berkeley.edu/docs/services/vhost/mail/">on our website</a>.<br />
<br />
<b>What if my student group already had mail virtual hosting?</b><br />
<br />
We've automatically migrated all of your existing forwarding addresses to the new system; there's nothing you need to do. In the future, you'll use the new web-based interface for modifying addresses instead of the <span style="font-family: Courier New, Courier, monospace;">.forward</span> files.<br />
<br />
<b>Can I forward to multiple addresses?</b><br />
<br />
Yup, each address can forward to as many other addresses as you'd like!<br />
<br />
<b>Can I have a catch-all address at my domain?</b><br />
<br />
Yes, and the web interface will help you set this up.<br />
<br />
<b>How do I get mail virtual hosting for my group?</b><br />
<br />
If you already have virtual hosting, then we can easily add mail service. If not, we can help you set up both a website and mail hosting.<br />
<br />
For details on our web virtual hosting service, see <a href="https://www.ocf.berkeley.edu/docs/services/vhost/">here</a>.<br />
<br />
For details on adding mail virtual hosting, see <a href="https://www.ocf.berkeley.edu/docs/services/vhost/mail/">here</a>.
