# Backup server compromised
<div dir="ltr" style="text-align: left;" trbidi="on">
We discovered that our backup server (pollution.ocf) had been recently misconfigured and was subsequently compromised. pollution maintains copies of account data (home directories, web directories, MySQL databases, and mail inboxes) and administrative credentials.<br />
<br />
We immediately investigated, changed the affected credentials, and notified campus. <b>Please assume that the attackers had access to a copy of your account data</b>, and take appropriate measures, especially if you store sensitive or restricted data. Although we determined that account credentials (e.g., password hashes), which are not stored on pollution, were not compromised, feel free to reset your password as an additional security measure.<br />
<br />
We sincerely apologize for this inconvenience. If you need any assistance, please let us know.</div>
