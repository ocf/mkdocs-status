# File storage system (NFS) restructure
We will be restructuring our file storage system sometime over the next semester. Please be aware that the u1-u8 file structure will be gone. If you have any files that depend on that file structure, they will not work anymore. They can be fixed by changing them to use your home directory (e.g. /home/f/fo/foo instead of /mnt/u3/home/f/fo/foo).<br />
