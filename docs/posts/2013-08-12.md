# Shorter URLs for web hosting
In our <a href="http://status.ocf.berkeley.edu/2013/07/ocfberkeleyedu-now-points-to-web-server.html">last blog post</a>, we announced that ocf.berkeley.edu points to the web server, meaning that you can now leave off the "www" in www.ocf.berkeley.edu.<br />
<br />
We're now announcing <b>ocf.io</b> as a shorter alternative to ocf.berkeley.edu, "in beta".<br />
<br />
In addition, if you leave off the tilde ("~") for an account name, it will be added.<br />
<br />
In other words, the following are shorthand forms of the canonical form http://www.ocf.berkeley.edu/~account :<br />
<ul>
<li><b>ocf.io/account</b></li>
<li>ocf.io/~account</li>
<li>ocf.berkeley.edu/account</li>
<li>ocf.berkeley.edu/~account </li>
<li>www.ocf.berkeley.edu/account </li>
</ul>
The shorthand URL forms are 302 redirected ("moved permanently") to the canonical URL.
