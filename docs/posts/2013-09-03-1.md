# Shell rc file changes
Due to upgrades of our servers to better supported systems, we will not be supporting old rc file locations. Please update your shell rc files accordingly. Please <a href="http://wiki.ocf.berkeley.edu/contact/">contact us</a> if you have any questions or concerns.<br />
<br />
The list below shows where the new files are located. (e.g. /folder/a --> /folder/b means /folder/a is now located at /folder/b) <br />
<br />
/opt/local/environment --> /opt/ocf/share/environment<br />
/usr/local/environment --> /opt/ocf/share/environment<br />
/opt/ocf/environment --> /opt/ocf/share/environment<br />
/usr/ucb/whoami --> /usr/bin/whoami
